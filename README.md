Project UAS Mata Kuliah Pemrograman Web menggunakan PHP Native, MySQL, dan AdminLTE.

- Dashboard

- MySQL

- AdminLTE
- DataTables

2. Import database `db_toko.sql`
3. Jalankan XAMPP
4. Buka localhost
