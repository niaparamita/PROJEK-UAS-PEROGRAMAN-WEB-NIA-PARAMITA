# PROJEK UAS PEMROGRAMAN WEB

## Deskripsi
Project UAS Mata Kuliah Pemrograman Web menggunakan PHP Native, MySQL, dan AdminLTE.

## Fitur
- Login
- Logout
- Session
- Dashboard
- CRUD User
- CRUD Produk

## Tools
- PHP
- MySQL
- XAMPP
- AdminLTE
- DataTables

## Cara Menjalankan
1. Clone repository
2. Import database `db_toko.sql`
3. Jalankan XAMPP
4. Buka localhost
